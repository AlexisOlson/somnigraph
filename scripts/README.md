# Scripts

## Sleep consolidation
- **sleep_nrem.py** — Offline batch processing: cluster similar memories, merge duplicates, detect contradictions, build graph edges.
- **sleep_rem.py** — Gap analysis and question generation: identify missing connections, surface unasked questions, generate reflective prompts.
- **sleep_timing.py** — Timing statistics for sleep runs.

## Research
Experiment scripts used during development. Not polished, but functional for reproducing results documented in `docs/experiments.md`.
